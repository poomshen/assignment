1. SpringMVC WebSite 생성
2. 필요한 Jar 파일 참조 (web , SpringMvc)
3. web.xml 
   -org.springframework.web.servlet.DispatcherServlet
   -<servlet-name>dispatcher</servlet-name>
   -<url-pattern>*.htm</url-pattern>
4. DispatcherServlet 을 통해서 Hanlder-Mapping 작업
   DispatcherServlet 사용하는 DI 작업 > [dispatcher]-servlet.xml 설정

5. [dispatcher]-servlet.xml 
   -안에서 bean 생성  
   -bean injection 하는 작업
   
6. SpringMVC (Model , View , Controller)
   -Model(DTO , VO , Domain)
   -View (jsp > el & jstl)
   -Controller 변화 : 
   --기본적으로 org.springframework.web.servlet.mvc.Controller  구현
   --public ModelAndView handleRequest 강제 구현
   --단점 : 게시판 목록  -> 클래스안에  함수하나가 요청 처리    
   --요청이 목록, 글쓰기 , 삭제 -> Controller 구현 클래스 3개  각 함수 기능처리
      
7. SpringMVC Annotation 사용해서 처리하자
   -*HandlerMapping : [함수 단위]로 처리
   -@Controller 
      @RequestMapping(/list.do)
       listBoard()    
      @RequestMapping(/listDetail.do)
       listDetail()
      @RequestMapping(글쓰기)
       boardWrite()
 
 8. 실습 기본 구성
    -com.controller ... 패키지 생성
    -view 폴더 : WEB-INF > view 폴더 생성
    -default 파일 : WebContent > index.html
